Что внутри

Bronze

raw_app_events — кликстрим: все события пользователей (page_view, add_to_cart, purchase и др.)
raw_orders — заказы со статусами, суммами и каналами привлечения

Silver

fact_sessions — агрегированные сессии: длительность, page views, bounce, конверсия
dim_channels — справочник каналов привлечения с бюджетами
Data Quality запросы: completeness, timeliness, аномалии в объёме данных

Gold

mart_daily_revenue — ежедневная выручка по каналам (GMV, NMV, AOV, уникальные покупатели)
dict_channels — словарь каналов для обогащения без JOIN

Продуктовая аналитика

DAU / WAU / MAU и Stickiness (DAU/MAU)
Когортный анализ — retention matrix по месяцам первого заказа
